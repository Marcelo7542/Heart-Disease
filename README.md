Predição de Doença Cardíaca com Regressão Logística
Sobre o Projeto
Neste projeto, desenvolvi um modelo de Machine Learning para prever a presença de doença cardíaca utilizando dados clínicos de pacientes do Heart Disease Dataset disponível no Kaggle.
O modelo foi construído utilizando Regressão Logística combinada com padronização dos dados por meio do StandardScaler.

O que foi feito:
Carregamento e preparação dos dados com Pandas.
Divisão em conjuntos de treino e teste.
Padronização das variáveis utilizando StandardScaler.
Construção de um Pipeline para pré-processamento e treinamento.
Busca dos melhores hiperparâmetros da Logistic Regression.
Avaliação utilizando validação cruzada com ROC AUC.
Geração de Classification Report.
Construção da Matriz de Confusão para análise dos resultados.
Tecnologias Utilizadas
Python
Pandas
Scikit-Learn
Matplotlib

Dataset:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

Objetivo

Aplicar técnicas de classificação supervisionada para identificar pacientes com maior probabilidade de apresentar doença cardíaca e avaliar o desempenho do modelo utilizando métricas apropriadas para problemas de classificação.
